# Release v0.0.2

README and publishing metadata update for Computer Care Skills.

## Changes

- Added Skills CLI installation commands.
- Documented how this repository is discovered by `skills.sh`.
- Added MIT license metadata.

## Install

```bash
npx skills add VintLin/computer-care-skills
```

List available skills:

```bash
npx skills add VintLin/computer-care-skills --list
```
